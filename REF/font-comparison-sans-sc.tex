{\LARGE\bfseries Sans Serif Font Comparison}\par
This reference sheet shows examples of sans-serif fonts that have usable small-caps functionality in this LaTeX setup.\par
Each font title is rendered in its own font, and each sample compares normal text, regular caps, and small caps.\par

\vspace{0.6em}
{\large\bfseries How To Use In Resume Files}\par
Your resume class defaults to Cabin. To override font in an individual resume, add one of the following lines in that resume preamble (after \verb|\documentclass|, before \verb|\begin{document}|):\par
\verb|\resumesetfont[sfdefault]{cabin}|\par
\verb|\resumesetfont[sfdefault]{roboto}|\par
\verb|\resumesetfont[sfdefault]{sourcesanspro}|\par
